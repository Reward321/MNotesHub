Introduction

This chapter deals with transition elements (d-block) and inner transition elements (f-block). These elements show unique properties like variable oxidation states, coloured compounds, paramagnetism, catalytic behavior, and complex formation.

Key Definitions

  • Transition Elements: Elements having partially filled d-orbitals in ground or excited state.
  • d-block Elements: Elements where last electron enters (n‚ąí1)d orbital.
  • f-block Elements: Elements where last electron enters (n‚ąí2)f orbital.
  • Lanthanides: 4f filling elements (Ce‚ÄďLu).
  • Actinides: 5f filling elements (Th‚ÄďLr).
  • Lanthanide Contraction: Gradual decrease in size across lanthanides.
